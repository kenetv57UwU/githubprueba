¿que es github?

GitHub es una plataforma en línea que se utiliza para gestionar proyectos de desarrollo de software utilizando el sistema de control de versiones Git. Fue fundada en 2008 y ha ganado una gran popularidad en la comunidad de desarrollo de software. GitHub proporciona una serie de herramientas y características que permiten a los desarrolladores colaborar en proyectos de código abierto o privados.
